# BioQuiz

Проект на Django для создания и прохождения викторины по биологии.

## Что делает проект
- Хранит вопросы в файле `questions.csv`
- Показывает список вопросов
- Позволяет добавить новый вопрос через веб-форму
- Запускает тест с перемешанными вопросами
- Сохраняет результаты в `results.csv`
- Показывает последние результаты для каждого пользователя

## Структура проекта
- `manage.py` — точка запуска Django
- `djangoapp/` — настройки проекта Django
- `quiz/` — приложение с представлениями, URL и шаблонами
- `questions.csv` — вопросы викторины
- `results.csv` — результаты прохождений

## Как запустить
1. Установить зависимости из `requirements.txt`
2. Запустить сервер:

```bash
python manage.py runserver
```

Открыть в браузере `http://127.0.0.1:8000/`.
